Share-A-Cart for Amazon.com

Version changes!

Below is the list of things that changed in this version of Share-A-Cart. Enjoy!

Name Change

The extension is now going to be called Share-A-Cart for Amazon instead of Amazon Share-A-Cart.

Receiving Carts

When receiving a cart, the extension will now wait until all of the items are in your shopping cart before loading the Amazon cart.

Options

You are now able to toggle if you want your shopping cart to be emptied when "Sending" it to a freind. That should save you some clicks. To toggle this option, please go to options tab in the help page. Please note that this is an experimental option and may have some issues (like missing an item or two)

Support for very large carts

If you have tried to send a very large cart (100+ items), you would have some issues. This is now possible with this release. Extension will now support up to 350 unique items in your shopping cart (quantity does not matter)
